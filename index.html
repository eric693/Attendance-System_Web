傳統打卡 vs 線上打卡比較

傳統打卡鐘

打卡方式
紙卡打卡鐘：

將紙卡插入機器，在卡片上留下日期時間印記。

感應式打卡鐘：

利用內建晶片的感應卡感應打卡，資料同步儲存於系統。

透過網路或應用程式完成考勤打卡，通常使用手機、電腦等裝置，進行 IP 或 GPS 位置驗證後即可打卡。

優點
  • ✓ 操作簡單且費用相對較低
  • ✓ 員工熟悉打卡流程
  • ✓ 設備購買即可使用
  • ✓ 不受地點限制，隨時隨地打卡
  • ✓ 資料即時同步至雲端系統
  • ✓ 出勤異常自動識別與提醒
  • ✓ 可設定指紋打卡機等打卡設備
  • ✓ 可整合 LINE 進行打卡
缺點
  • ✗ 紙卡打卡需人工整理統計
  • ✗ 無法處理遠距辦公、外勤等狀況
  • ✗ 容易忘記打卡或代打卡
  • ✗ 仰賴網路環境連接
  • ✗ 可能發生忘記打卡情況

常見問題

LINE 打卡的設定會很麻煩嗎？

系統在初始時會協助建好基本設定，並提供教學影片。不過因各公司規則不同，部分細節仍需自行調整。若在操作上有任何疑問，歡迎與我們客服聯繫，我們有專人協助，讓您快速上手、無需煩惱！

適合我們公司使用嗎？

當然，出勤管家適合所有規模的公司。無論你是 5-10 人的小微企業，或是 500 人以上的中大型企業，只要你想用系統降低人事作業時間，並且讓員工在 LINE 上完成一切，出勤管家都是你的最佳選擇。

有使用人數限制嗎？

不限人數。我們推出的最新限量方案：手機打卡、LINE 打卡不限使用人數，並且是買斷制的付費方式，非常優惠先搶先贏。（BC方案適用 50 人以下，如 50 人以上，歡迎與我們聯繫）

購買之後，該怎麼開始使用？

請於線上選擇要購買的方案，並且完成付款。我們將於 1-2 工作天內與您聯繫，並且協助你設定好公司規則、讓員工直接開始使用。流程說明：完成購買 → 專人設定規則/系統教學 → 開通帳號 → 員工開始使用。

有什麼打卡方式可以選擇？

打卡方式有 GPS、IP 可選擇。你可以選擇 GPS 定位打卡，員工抵達指定地點後才可以進行打卡。或是選擇 IP 打卡，需連上指定網路後，員工才可以進行打卡。

怎麼防止員工還沒到公司就打卡？

你可以設定使用 GPS 定位打卡或 IP 打卡，員工抵達公司、連結上公司網路後才可以進行打卡。

怎麼防止員工代打卡？

綁定員工的 LINE 進行打卡，搭配系統機制，就可以輕鬆避免員工代打卡的情況！

是否適用彈性上下班時間的情況？

適用！很多企業都有實施彈性上下班機制，出勤管家可以讓你自由設定彈性上下班的時間區間，自動依據員工上班時間推算出當日應下班的時間。也可以依據設定自動計算遲到、早退等出勤異常情況。非常方便！

可以設定多組打卡規則嗎？

可以！出勤管家可以讓你設定多組打卡規則，並套用在適合的員工。例如：高階主管的打卡規則、一般員工的打卡規則，或是門店長的打卡規則、可以遠距辦公的打卡規則等。

如何使用工作日誌功能？

工作日誌功能非常簡單：

員工端：
1. 登入系統後進入「工作日誌」分頁
2. 選擇日期並填寫工作時數
3. 詳細記錄當日工作內容（至少 10 個字）
4. 提交後等待主管審核

主管端：
1. 在管理員介面查看待審核日誌
2. 檢視員工的工作內容與時數
3. 可選擇核准或拒絕，並填寫審核意見
4. 系統會自動通知員工審核結果

提示：被拒絕的日誌可以修改後重新提交

我們的 AI OCR 發票辨識系統具有以下特點：

高準確度：

我要怎麼查詢員工遲到幾分鐘？

在出勤報表中，已經自動幫你計算員工每天遲到幾分鐘，可以輕鬆加總遲到分鐘數。再也不用手動計算了！

可以看到即時的出勤資料嗎？

可以！後台的出勤報表可以看到最即時的出勤狀況。不同辦公地點也可以在同一個後台一目瞭然。

可以看到員工每日實際上班的工時嗎？

可以！出勤報表中，會自動幫你計算員工每日上班時數。這張報表也可以方便公司計算薪資。

可自動判斷遲到、早退的時間嗎？

可以！出勤報表中，會幫你計算員工每日出勤狀況，包含遲到、早退，以及自動計算遲到了幾分鐘、早退了幾分鐘。這張報表也可以方便公司計算薪資。

員工請假的時候，可以被審核嗎？

可以！你可以設定是否需要審核、給誰審核。可以設定給部門主管審核或是特定的審核人，也可以設定要審核幾關卡。

可以匯出工作日誌報表嗎？

當然可以！工作日誌報表功能非常完整：

個人報表：
• 員工可匯出自己的工作日誌
• 選擇月份即可下載 Excel 檔案
• 包含日期、時數、內容、審核狀態

管理員報表：
• 可匯出單一員工或全部員工的日誌
• 支援自訂日期區間
• 報表包含員工姓名、部門、工作統計
• 可用於績效評估或專案追蹤

提示：所有報表都以 Excel 格式匯出，方便後續編輯或分析
